Ev3 in Relation to the Computer

1. Identify the different parts of the EV3 computer

A computer has four parts,, and they are:
Input, Output, Processor, and Storage. In the Ev3, the input is the sensors, the output is what the robot does based on the input, the processor is the programmable brick, and the storage is the memory chips inside the brick. Which brings us to our next question.


2. Describe the role of the different types of memory chips in the EV3 computer

There's RAM and FLASH memory in the EV3 brick.
RAM stands for Random Access Memory. FLASH is discs that actually spin.
Flash is used for all memory except the memory used when actually running the program.
RAM is used only when the program is running.
